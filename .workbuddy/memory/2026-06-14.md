## Swift 6 并发编译错误修复 (13:46 - 13:50)

### 问题描述
用户反馈 Xcode 编译报错，截图显示：
1. **WaterRecordStore 错误**：
   - `Main actor-isolated static property 'storeURL' cannot be accessed from outside of the actor`
   - `Reference to captured var 'self' in concurrently-executing code`
2. **Assets 警告**：`The app icon set "AppIcon" has an unassigned child`
3. **其他警告**：Update Widget Data script、recommended settings

### 修复内容
**Stores/WaterRecordStore.swift**:
- `storeURL` 添加 `nonisolated` 修饰符（允许后台线程安全访问）
- `save()` 方法改为先快照 `items` 再传入 `performSave(items:)`（避免 DispatchWorkItem 中捕获 self）
- `load()` 改用 `Task { @MainActor in }` 替代 `Task.detached`（消除 self 并发捕获）

**Assets.xcassets**:
- 删除未引用的文件 `A_premium_iOS_app_icon_for_a_w_2026-06-14T03-02-18.png`（AI生成的原始带水印图片）

### 验证结果
- ✅ **BUILD SUCCEEDED**
